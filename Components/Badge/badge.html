Badge

Bagdes are small circles positioned on the top-left or top-right of the parent component to notify the new changes, They can be used to display numbers, online/offline status, etc.

Note: Parent element here is a span i.e an inline element which inherit the width according to the content.

Badges over Texts

You can use class badge to the parent element to position it relative and then badge-right | badge-left to the child element to get it positioned around the parent element.

text+7 text+7

<-- Badge on the top-right corner -->

<span class = "badge badge-span" > text
<span class = "badge-right badge-circle" > +7 </span>
</span>

<-- Badge on the top-left corner -->

<span class = "badge badge-span" >
<span class = "badge-left badge-circle" > +7 </span>
</span>
