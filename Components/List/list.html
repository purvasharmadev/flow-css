List

A list is a record of short pieces of related information or used to display the data or any information in web pages in the ordered or unordered form. GlowCss defines basic list style classes like list | | list-border | list-item | list-item-border | list-title | list-item-hover | list-item-background.

Note:By default the list takes 100% width of parent's full width. In the given example the text alignments, font sizes are done using utitlity classes to make classes customizable.
