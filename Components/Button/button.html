Button

Buttons allow user to take action or interact with a webpage. GlowCss offers wide range of buttons to suit your design.

Deault Button

Create a button with default styling using btn class.

<button class = "btn"> Default Button </button>

Solid Button

Create a solid button with btn-primary | btn-secondary | btn-danger | btn-info | btn-warning class.

<-- btn-primary -->

<button class = "btn btn-primary"> Primary Button </button>

<-- btn-secondary -->

<button class = "btn btn-secondary"> Secondary Button </button>

<-- btn-danger -->

<button class = "btn btn-danger"> Button </button>

<-- btn-info -->

<button class = "btn btn-info"> Button </button>

<-- btn-warning -->

<button class = "btn btn-warning"> Button </button>

Outline Button

Create a outline button with btn-primary-outline | btn-secondary-outline class.

<-- btn-primary-outline -->

<button class = "btn btn-primary-outline"> Primary Button </button>

<-- btn-secondary-outline -->

<button class = "btn btn-secondary-outline"> Secondary Button </button>

Link

Create a link button with link class.

link

<-- link -->

<a href = "#" class = "link"> link </a>

Icon Button

Create a Icon button with btn-icon class.

<-- btn-icon -->

<button class = "btn btn-icon"> <svg> </svg> </button>

<-- btn-icon with text -->

<button class = "btn btn-icon"> <svg> </svg> Home </button>

Float Button

Create a Float button with btn-float class.

<-- btn-icon -->

<button class = "btn-float"> <svg> </svg> </button>
