Modal

Modal windows are user interface elements. They sit on top of the main interface, so they do not interfere with underlying workflows. They are information boxes that pop-ups wihtout disturbing the rest of the screen. GlowCss allows you create different kind of modals.

Note: modal has z-index:1; and takes 100% width of parent div .
