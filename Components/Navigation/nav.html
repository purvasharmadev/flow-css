NavBar

NavBar allows user to navigate through webpages by grouping the links in a nav element. GlowCSS uses Nav semantic tag and gives classes like navbar-list | nav-item | nav-list | nav-brand | nav-link | nav-resp | nav-toggle to create Responsive navbar which looks good while serving the purpose.

Note: Resize the browser window to see responisve property.

Simple Responsive Navbar

BrandName

  • Home
  • About
  • Contact
<nav class = "bg-primary color-white" >
<div>
<svg class = "nav-logo img-responsive img-rounded"> </svg>
<a href = "#" class = "nav-link link" > BrandName </a>
</div>
<ul class = "nav-list">
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > Home </a> </li>
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > About </a> </li>
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > Contact </a> </li>
</ul> </nav>

Responsive NavBar with Toggle

BrandName

☰
  • Home
  • About
  • Contact


<nav class = "nav-resp position-relative bg-primary color-white" >
<h2>
<svg class = "nav-logo img-responsive img-rounded"> </svg>
<a href = "#" class = "nav-link link" > BrandName </a>
</h2>
<!-- Toggle , You can use #9776; for toggle icon -->
<span class = "nav-toggle"> ☰ </span>
<!-- Navbar List -->
<ul class = "navbar-list">
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > Home </a> </li>
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > About </a> </li>
<li class = "nav-item" >
<a href = "#" class = "nav-link link" > Contact </a> </li>
</ul> </nav>
