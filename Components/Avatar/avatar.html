Avatar

Avatar are used as user profile, they are usually circular in shape. GlowCss offers you two major types of avatar in different sizes.

Circular Avatar

avatar use this class inside of an img tag to make it behave like an avatar.
To get different sizes use: avatar-sm | avatar-md | avatar-lg ,

<img src = "img.png" class = "avatar avatar-sm" alt = "..." >
<img src = "img.png" class = "avatar avatar-md" alt = "..." >
<img src = "img.png" class = "avatar avatar-lg" alt = "..." >

Square Avatar

avatar-square use this class inside of an img tag to make it behave like an avatar.
To get different sizes use: avatar-sm | avatar-md | avatar-lg ,

<img src = "img.png" class = "avatar-sqaure avatar-sm" alt = "..." >
<img src = "img.png" class = "avatar-square avatar-md" alt = "..." >
<img src = "img.png" class = "avatar-sqaure avatar-lg" alt = "..." >
