Alerts

Alerts can be used to show or notify a short message to a user without interrupting the user's task.

Note: By default the alert component takes the full width of the parent element.

Alert Danger

Danger! Indicates a dangerous or potentially negative action.
<div class = "alert alert-danger"> Lorem ipsum dolor sit. <div>

Alert Sucess

Success! Indicates a successful or positive action.
<div class = "alert alert-sucess"> Lorem ipsum dolor sit. <div>

Alert Info

Info! Indicates a neutral informative change or action.
<div class = "alert alert-info"> Lorem ipsum dolor sit. <div>

Alert Warning

Warning! Indicates a warning that might need attention.
<div class = "alert alert-warning"> Lorem ipsum dolor sit. <div>

Dismissable Alert

Warning! Indicates a warning that might need attention. ×
<div class = "alert alert-info"> Lorem ipsum dolor sit. <span "alert-close-btn" onclick="this.parentElement.style.display='none';"> × </span> <div>

Alert with Outline

Warning! Indicates a warning that might need attention.

Info! Indicates a neutral informative change or action.

Success! Indicates a successful or positive action.

Danger! Indicates a dangerous or potentially negative action.

<div class = "alert alert-warning-outline"> Lorem ipsum dolor sit. <div>
<div class = "alert alert-info-outline"> Lorem ipsum dolor sit. <div>
<div class = "alert alert-sucess-outline"> Lorem ipsum dolor sit. <div>
<div class = "alert alert-danger-outline"> Lorem ipsum dolor sit. <div>
